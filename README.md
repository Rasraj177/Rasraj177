### 🌟 Featured Projects

**🌫️ [AirSense-AI](https://github.com/Rasraj177/AirSense-AI)** — Real-Time AQI Prediction Platform
Real-time AQI forecasting for Delhi using live Government of India API data from 30+ monitoring stations. End-to-end ML pipeline with a Random Forest Regressor for 24-hour & 7-day predictions, served via an interactive Streamlit + Plotly dashboard.
`Python` `Pandas` `Scikit-learn` `Streamlit` `Plotly`

**💳 [CreditLens](https://github.com/Rasraj177/CreditLens)** — AI-Powered Loan Risk Assessment System
Loan default risk prediction system trained on structured financial and demographic data. Handled class imbalance with SMOTE and optimized a Random Forest Classifier to **ROC-AUC = 0.865**, with a Streamlit dashboard surfacing risk bands and default probability.
`Python` `Scikit-learn` `SMOTE` `Streamlit`

**🧑‍💼 [TalEntry](https://github.com/Rasraj177/TalEntry)** — AI-Powered Smart Resume Screening & Candidate Ranking
NLP-driven candidate ranking system using Sentence Transformers, built during my AI internship at IBM PBEL.
`Python` `NLP` `Sentence Transformers`
